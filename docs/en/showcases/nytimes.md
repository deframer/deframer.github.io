## Trend Mining

<img src="/assets/screenshots/portal/nypost_tag_cloud.png" alt="The New York Times - Tag Cloud" width="1000"/>

- **Tag Cloud** we show a tag cloud with the most common talking points for the selected period

<img src="/assets/screenshots/portal/nypost_lifecycle.png" alt="The New York Times - Life Cycle" width="1000"/>

- **Lifecycle** if you select a trend item from the tag cloud, you see the lifecycle for the selected period

<img src="/assets/screenshots/portal/nypost_context.png" alt="The New York Times - Context" width="1000"/>

- **Context** if you select a trend item from the tag cloud, you see the context (verbs) how the item is used

<img src="/assets/screenshots/portal/nypost_vs_washington_post.png" alt="The New York Times - Compare" width="1000"/>

- **Compare** You can compare any know source for the selected language with any other source. You see the talking points of the current source, blind spots that your current source is not talking about and common talking points.
